## Tasks Completed

### 1. FizzBuzz (1–50)
- Prints numbers from 1 to 50
- Prints **"Fizz"** for multiples of 3
- Prints **"Buzz"** for multiples of 5
- Prints **"FizzBuzz"** for multiples of both 3 and 5

**File:** `fizzbuzz.py`

---

### 2. Number Guessing Game
- The computer selects a random number
- The user guesses the number
- Displays:
  - `"Too high"` if the guess is higher
  - `"Too low"` if the guess is lower
- Continues until the user guesses correctly

**File:** `guessing_game.py`
